# Security Policy

## Supported Versions
This repository currently supports security fixes for the latest version on the `main` branch.

| Version/Branch | Supported |
| --- | --- |
| main | ✅ |

## Reporting a Vulnerability
Please **do not** create public GitHub issues for security vulnerabilities.

Instead, use one of the following:

1) **Private Vulnerability Reporting (preferred)**  
Go to: **Security → Advisories → Report a vulnerability** (if enabled)

2) **Email (fallback)**  
Email: gavz2608@gmail.com  
Include:
- steps to reproduce
- impact
- affected files/versions
- any PoC (if safe)

## Response Targets
- Acknowledgement: within **48 hours**
- Initial assessment: within **5 business days**
- Fix / mitigation plan: as soon as reasonably possible

## Coordinated Disclosure
We appreciate responsible disclosure and will coordinate a public advisory after a fix is available.
